# Employee Task $ Approval Management System 

## Overview 
A backend-focused system buitl with Django and DRF to manage task assignment and approval workflows using role-based access control and JWT authentication

## Roles
- Admin: Manage Users and projects 
- MAnage: Assigns task and approves work 
- Employee: Work on assigned tasks

## Features
- JWT-based authentication 
- Role-based authorization
- Task life-cycle manmagement 
- Approval rejection workflow
- Secure and scalable backend design 

## Task stack
- Backend: Django, DRF
- Database: SQLite (development)
- Authentication: JWT
- Tools: Git, Postmen
